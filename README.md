**APP**         : BACKEND INSTALL
**FRAMEWORK**   : fastapi

Install package:
--------------
> sudo apt-get install python3 python3-venv python3-dev python3-minimal

Install application:
--------------------

1. Make virtual enviroment
> python3 -m venv venv

2. Source to library:
> source venv/bin/activate

3. Install package
> pip install -r requirements.txt

4. Runing  app:
> python main.py
